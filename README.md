# Customer Churn Analytics Platform

> An end-to-end analytics project predicting customer churn using Python, PostgreSQL, SQL, Machine Learning, and Power BI.

## Business Problem
Telecom companies lose 15-25% of customers annually. Retaining a customer costs 5-7x less than acquiring one. This platform identifies at-risk customers and provides actionable retention recommendations.

## Dataset
- Source: IBM Telco Customer Churn Dataset
- Records: 7,043 customers
- Features: 21 columns
- Target: Churn (Yes / No)
